# 宇树应用模板

欢迎使用宇树应用模板仓库！本仓库旨在为开发者提供一个**开箱即用**的环境，帮助您快速上手宇树机器人的应用开发。

## 仓库结构

本仓库采用 Monorepo 结构，以下是本仓库的主要内容。

### 示例项目

| 目录路径 | 示例名称 | 语言 | 主要功能描述 |
| -------- | -------- | ---- | ------------ |
| `g1_mimic_demo/` | G1 模仿学习演示 | `C++` 主控，`Python` 提供服务 | 展示如何使用模仿学习策略控制 **G1** 执行动作（如江南 Style、开场拳等）。 |

关于如何控制机器人的详细内容，您可以参考[宇树仓库](https://github.com/unitreerobotics) 中对应语言的 SDK。

### 产物约束

`App` 使用 `docker` 作为基础设施，支持任意语言和框架，支持任意 `Linux` 发行版，打包发布只需要满足以下**最小要求**：

| 文件/文件夹 | 是否必选 | 说明 |
| ----------- | -------- | ---- |
| `app/` | 必选 | 应用程序文件夹，包含程序运行所需的二进制或脚本，入口在 `metadata.yaml` 中配置，需通过 **80 端口**提供您自己的 `http` 服务 |
| `metadata.yaml` | 必选 | 应用元数据文件，包含应用基本信息及配置 |

## 快速开始

### 运行 G1 模仿学习演示

进入目录并启动服务：

```bash
cd g1_mimic_demo
python3 ./app/g1/cmd_server.py
```

详细说明请参考 [`g1_mimic_demo/README_zh.md`](./g1_mimic_demo/README_zh.md)。

### 打包

```bash
tar -czvf your_app_name.tar.gz ./app
```

之后将应用 `your_app_name.tar.gz` 上传到 UniStore 即可。
